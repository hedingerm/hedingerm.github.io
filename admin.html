Anleitung
Bildverwaltung:
  • Wählen Sie die gewünschte Kategorie aus den Tabs
  • Ziehen Sie Bilder in den Upload-Bereich oder klicken Sie zum Auswählen
  • Bilder werden automatisch auf optimale Größe angepasst
  • Verwenden Sie aussagekräftige Dateinamen für bessere Organisation
Empfohlene Bildgrößen:
  • Bier-Bilder: 400x400px (quadratisch)
  • Galerie: 800x600px (Querformat)
  • Schankwagen: 800x600px (Querformat)
  • Brauerei: 800x600px (Querformat)
  • Team: 600x400px (Querformat)
Wichtig: Stellen Sie sicher, dass Sie die Rechte an allen hochgeladenen Bildern besitzen.
